## How to Build

1. [node.js](https://nodejs.org/ja/)をインストールします
2. リポジトリをダウンロードします

```
npm i
npm i grunt-cli node-gyp -g
grunt
```

## How to Customize yourself

```
grunt server
```
を実行した後、 https://localhost:3000/maqz/ を参照してください。
