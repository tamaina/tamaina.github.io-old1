# 反省
1. 相互審判のはずなのに、審判を出さないクラスがあった
2. トーナメント表に結果を書くべきだった
3. バレーやバスケと比べてしまうと、やはり盛り上がりに欠ける部分があった
4. 結果を出しに行く際に体育館競技には1つも無線がなかったこと

# 改善方法
1. 声をかける
2. どこが勝ち上がっているか目立つように書く
3. 第一体育館をもっと広く使い、応援できるスペースを増やす
4. 無線を準備し、結果をすぐに伝えられるようにする。
