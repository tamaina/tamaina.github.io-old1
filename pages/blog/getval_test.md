## この記事の意味

maqzでは、＄{~}を利用して特定の内容を代入することができます。

eval関数の中で、HTML変換後の文字列をテンプレートリテラルとして展開しています。

## 見本

この記事のソースのmd5ハッシュ値は"${page.meta.md5}"です。

```
この記事のソースのmd5ハッシュ値は"\${page.meta.md5}"です。
```

2かける3は${ 2 * 3 }です。

```
2かける3は\${ 2 * 3 }です。
```
